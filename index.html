About Me!

I am a computer science major with a HCI specialization at the University of Rochester. Currently I am a Sophomore and am taking classes in Artificial Intelligence and Web Design, both of which I am very interested in! I am particularly interested in making advanced and sophisticated technologies accessible to everyone so that their impact can be even greater. I am looking for a internship this summer in the computer science field and also hoping to get involved in HCI research soon! My hobbies include digital design, crafting, snowboarding, watching documentaries, learning about the world around me, and more.

Projects I've worked on:

  • Connect Four AI: My most recent project for Artificial Intelligence was creating a connect four AI that plays well enough to beat humans. It was a really exciting project because I find the minimax algorithm along with other search methods to be extremely interesting. It is also a great feeling to write a program that is better then I am at connect four.
  • In Computation and Formal Models I completed many projects including creating DFAs, NFAs, Recursive and table driven parsers for grammars, functional programming, implementing our own relational data model, and more. This was the first theory heavy computer science class I took and it was challenging but I found it very interesting and a very rewarding class to take.
  • In Data Structures and Algorithms I worked on projects ranging from using Dijkstras algorithm to find the shortest path between any two points in New York state, creating 2048, creating a calculator that functions the way using python as a calculator would, and using Huffman encoding to compress a image and then decompress it. The Huffman coding project was my favorite because it was really cool to be able to take a file, encode it then decode it and have the same file again!
  • I have also made a website with a partner that took a huge set of data and helped people find the closest farmers market to them with lots of information about the farmers market and had the ability to search for specific things. We did this website for the Young Rewired State of Code hackathon and ended up getting first place in the international category.
  • I have also worked on creating versions of fix it felix (in java) and frogger (in javascript)
  • I made a website in high school that used another large data file and allowed the user to search up any restaurant in NYC and see its history of health code ratings and violations. It was one of my first more serious computer science projects and I am still proud of it!
  • I am currently working on an app as a project manager and UX designer. More about this project will be added as it develops.
